# Portfolio 2025

What I build this year...

## 1. Video Streaming iOS app (swift)
https://github.com/fiveoceans-dev/NounsTV
NounsTV is an iOS app for watching nounish films from youtube. It primarily uses Swift for development, with an iOS app structure managed via Xcode.

## 2. Social Voting iOS app (swift, ethereum)
https://github.com/fiveoceans-dev/ios-dao-app
NounsDAOApp is an iOS social voting app integrated with Ethereum. It uses Swift, NounsAPI, MetaMask iOS SDK, and Apollo iOS for GraphQL operations.

## 3. Programming competition website (typescript, nodejs, PostgreSQL)
https://github.com/fiveoceans-dev/solidity-masters
Solidity-masters is a Solidity programming competition platform. It uses JavaScript, TypeScript, EJS, CSS, and Solidity for blockchain elements. The stack includes Next.js for frontend/server-side rendering, Prisma for database management, and Supabase PostgreSQL support.

## 4. ATM project (C++)
https://github.com/fiveoceans-dev/atm_project
Implement functions, classes, methods for development of ATM machine. The project mainly for study purposes and uses C++.

## 5. Remote Control based on webRTC ()
https://github.com/fiveoceans-dev/beam_robotics
Beam Robotics is a vendor-agnostic remote command and control platform for robotics, including components for CAD, simulation, systems, templates, and web. The dev stack involves multi-language development with directories for various robotics aspects. Programming languages used: ASP.NET, C, C++, JavaScript, CMake, CSS, and Other.

## 6. Cluster of GPUs ()
https://github.com/fiveoceans-dev/nextmetal
Nextmetal is a web and macOS project for GPU cluster management. It uses TypeScript, EJS, CSS, JavaScript, Swift, PLpgSQL, and HTML.

## 7. Online Sailing Game
https://github.com/fiveoceans-dev/regatta-mix
Online Sailing Game Regatta Rift (regatta-mix) is a virtual sailing racing platform. It uses TypeScript, PLpgSQL, CSS, and Other. The dev stack is a monorepo with Vite/React/TypeScript frontend (shadcn-ui, Tailwind CSS), Supabase backend (edge functions, multi-site schemas), and Vercel deployment.

## 8. Stock Trading app
https://github.com/fiveoceans-dev/hypermetal
Hypermetal is a stock trading platform as a Yarn Berry monorepo with frontend, backend, CLI, and on-chain integrations. It uses TypeScript, Solidity, CSS, HTML, JavaScript, Shell, and Dockerfile. The stack includes Vite/React for frontend, Express for backend, Yarn for dependencies, Docker for containerization, and SDKs for Hyperliquid, Pyth, and deBridge.

## 9. Online Newspaper
https://github.com/fiveoceans-dev/open-air-chronicles-react
Open Air Chronicles is a digital newspaper with responsive layout, articles, user roles, and photography stock. It uses TypeScript (95.1%) and CSS (4.2%). The stack includes React 18, Tailwind CSS, Shadcn UI, Lucide React icons, React Router DOM, Vite for build, and Supabase for backend.

## 10. Online Poker
https://github.com/fiveoceans-dev/pokernft
PokerNFT is a real-time multiplayer Texas Hold'em poker platform with Starknet integration. It uses TypeScript (83.9%), Cairo (14.9%), Shell, CSS, JavaScript, and Python. The stack includes Next.js for frontend (on Vercel), Yarn workspaces, Docker for backend, WebSocket server (on Heroku), and Cairo for smart contracts.

## 11. Online store
https://github.com/fiveoceans-dev/buena
Buena is a simple online store ready for AI agents. It uses TypeScript, PLpgSQL, CSS, and Other. The dev stack includes React/Vite/TypeScript for frontend, Tailwind CSS, shadcn-ui components, and Supabase for backend/database.
